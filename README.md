# E-Safaai
E-Safaai an android project for digital complaining
A MIT-ADT Persona'19 Hackathon Winner Android application for solving the environmental issues in society or campus like water leakage, garbage or open pot holes.
User can click the picture of the issue and submit it to the local corporator. The image will be sent with the real time location of user with Google Maps link, and additional description about issue, so that corporation can take immediate action on the issue to solve it.
Once issue is submitted, the copy of image and location link will be saved on Firebase real time storage. Users can monitor the total number of issues in the area or the city, by category in the dashboard which is developed using R and Shiny Dashboard.

Technologies Used : Android, Firebase, R programming
